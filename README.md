# ktnaOS & ClackOS

A collection of system-level Spicetify extensions and themes for full control.

## Included Extensions
- **ktnaOS**: Terminal UI, Dashboard, and Theme Swapper.
- **ClackOS**: Dashboard and Theme Swapper.
- **Clack Visualizer**: Audio visualizer overlay.

## Included Themes
- **ktnaOS Themes**: Collection of 20+ dark and cyberpunk color schemes.
